O ClimaJá é um projeto desenvolvido para o curso de [FavelaWare] focado no Objetivo de Desenvolvimento Sustentável (ODS) 13: Ação Contra a Mudança Global do Clima.

O site interativo e responsivo visa conscientizar e engajar o cidadão comum, traduzindo o risco climático global em ações e metas pessoais e locais. A plataforma combina dados de clima em tempo real com ferramentas de simulação de impacto, permitindo que cada usuário visualize como suas pequenas mudanças diárias contribuem para o objetivo maior.
